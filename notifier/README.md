# notifier.sh

`notifier.sh` is a simple shell script that hooks onto a given process and monitors it. If the process is done, you get a notification (per e-mail).



### Usage

```bash
sh ./notfier.sh [process]

eg.: sh ./notfier.sh ssh
```
